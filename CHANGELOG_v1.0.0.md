# 更新日志

## v1.0.0 - 首个 tkinter 重构版本

基于 ZiHaoSaMa66 原版 OsEasy-ToolBox（Flet 框架）全面重构为 Python tkinter 原生版本。

### 更动

- **UI 框架替换**：Flet → 原生 Python tkinter，去掉 WebView 依赖，体积大幅减小
- **Python 版本兼容**：支持 Python 3.8+
- **系统托盘**：点击关闭按钮最小化到系统托盘（需 pystray 依赖），右键可显示/退出
- **自动提权**：启动时自动通过 PowerShell 请求管理员权限
- **彩色托盘图标**：自定义蓝色工具箱图标

### 新增功能

- **拦截教师端远程重启**：通过映像劫持 shutdown.exe 实现，防止教师远程重启电脑
- **系统托盘支持**：点击 ✕ 最小化到托盘，后台静默运行，右键菜单显示/退出
- **粘滞键后门**：注册后连按5次 Shift 调出工具箱，即使在锁定/广播状态下也能操作
- **解锁方案二选一**：传统文件删除 或 映像劫持（不删文件，可还原）
- **高版本学生端自动关 MMPC 服务**：适配新版噢易客户端
- **全功能确认弹窗**：每个危险操作都有确认对话框，防止误触

### 注意事项

- 打包后的 exe 可能被杀毒软件误报（PyInstaller 特征），建议添加白名单
- 本工具仅供学习和技术研究使用，请遵守机房/学校规定
